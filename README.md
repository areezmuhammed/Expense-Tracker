# Monthly Expense Tracker & Saver üí∞

A modern, premium-designed web application to track your monthly expenses, visualize your spending habits, and receive smart savings recommendations. Built with **React**, **Vite**, and **TailwindCSS**.

## üöÄ Features

-   **Transaction Logging**: Easily add daily expenses with categories (Food, Transport, Utilities, etc.).
-   **Interactive Dashboard**: Visualize your spending breakdown with dynamic Pie Charts.
-   **Smart Savings Engine**: Rule-based logic that analyzes your spending to provide actionable saving tips (e.g., *"You've spent 40% of your budget on Food"*).
-   **Persistent Data**: Your data is saved locally in your browser, so you never lose track.
-   **Responsive Design**: A sleek, dark-mode compatible UI that works on desktop and mobile.

## üì¶ Installation & Setup

Prerequisites: Node.js installed.

1.  **Clone the repository**
    ```bash
    git clone https://github.com/areezmuhammed/Expense-Tracker.git
    cd Expense-Tracker
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Run the App**
    ```bash
    npm run dev
    ```
    *Note: If you are on the original developer's machine with custom paths, use `.\run-app.bat`.*
